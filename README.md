# Deep File Search Engine #

### What is this repository for? ###

* Google for local files
* Quick search across files. Mac like file searching for linux/windows.
* Version 0.1

### How do I get set up? ###

* Install mongodb
* Install python
* Install pip
* pip install pyinotify
* run mongod
* run src/deamon.py
* run src/query_builder.py
* enjoy...!
